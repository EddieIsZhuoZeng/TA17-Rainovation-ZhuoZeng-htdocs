== Description ==
Highly Secure and Powerful WordPress File Manage with Multiple Stunning Designs.

You can **Create, Delete, Edit, Download, Upload, Compress (Zip)** files without any FTP software. You can manage all your website files right from your WordPress Dashboard.

You can also connect to **Remote FTP location** and copy files/folders from there to your server without any trouble.

You can define the **ONLY** user who can access the File Manager for higher security.

**Core Features:**

* **Create Files and Folders**
* **Delete Files and Folders**
* **Edit Files and Folders**
* **Download Files and Folders**
* **Upload Files and Folders**
* **Compress Files and Folders**
* **Preview Files and Folders**
* **Drag-n-Drop Files and Folders**
* **Select Multiple Files and Folders**
* **Inverse Selection**
* **Get Info of Files and Folders**
* **List View**
* **Sort Files and Folders**
* **Full Screen Support**
* **Access Root Directory**
* **Access to WordPress Configuration Files**
* **Multisite Compatible**

== Installation ==
1. Download the plugin zip file.
2. Extract the downloaded zip file.
3. Upload the \"wp-secure-file-manager\" directory to the \"/wp-content/plugins/\" directory.
4. Activate the plugin through the \"Plugins\" menu in WordPress.
